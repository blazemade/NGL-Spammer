# NGL-Spammer
NGL-Spammer is a script made using python to spam specific accounts with messages on NGL.

To run it on Windows based systems:
1) Download the repo
2) install requirements
3) run the .bat file

For linux based systems:
1) clone the repo
2) install the requirements
3) run the script

This script uses the follwing modules:
1) Request (duh)
2) uuid
3) time 
4) os
